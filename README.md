## Portfolio Website

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This project is a changing collection of records that reflect my accomplishments, skills, experiences and attributes. It highlights and showcases samples of some of my work, along with life experiences, values and achievements.

## Highlight Features
- User-friendly navigation
- Intuitive design and user experience
- Responsive web design - adapted to a variety of devices available on Google Chrome Developer Tools
- Clear calls to action
- Animations and smooth transition effects

## Note
- Recommended devices and web browser to visit this website: 17'' Dell Laptop, Samsung Galaxy S7 and Google Chrome
